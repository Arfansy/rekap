HTML CSS

20 Juli 2025 Oleh Admin Kategori: Kelas 10

HTML adalah bahasa markup standar yang digunakan untuk membuat struktur dasar dari sebuah halaman web. HTML tidak bersifat pemrograman, melainkan markup yaitu memberi tanda atau label pada konten seperti teks, gambar, tombol, dan lain-lain

Sedangkan CSS adalah bahasa yang digunakan untuk mengatur tampilan dan gaya dari elemen-elemen HTML. Dengan CSS, kamu bisa mengatur warna, ukuran, font, jarak, tata letak, dan bahkan animasi

Fungsi HTML dan CSS

Fungsi HTML:

  • Membuat struktur halaman web (judul, paragraf, gambar, tautan, tabel, dll)
  • Menentukan elemen konten seperti header, footer, article, form, dll
  • Mengatur posisi dan urutan konten
  • Menjadi kerangka utama dari sebuah website, yang kemudian dihias dengan CSS dan difungsikan dengan JavaScript

Fungsi CSS:

  • Membuat struktur halaman web (judul, paragraf, gambar, tautan, tabel, dll)
  • Menentukan elemen konten seperti header, footer, article, form, dll
  • Mengatur posisi dan urutan konten
  • Menjadi kerangka utama dari sebuah website, yang kemudian dihias dengan CSS dan difungsikan dengan JavaScript
